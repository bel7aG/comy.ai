# Started boilerplate for ai

A modern, production-ready Next.js starter template built with best practices and developer experience in mind.

## Features

- **Next.js 14+** - App Router, Server Components, and all the latest features
- **TypeScript** - Full type safety across the entire codebase
- **Tailwind CSS** - Utility-first styling with custom theme support
- **Turborepo** - Monorepo setup for scalable project structure
- **ESLint & Prettier** - Consistent code formatting and linting
- **pnpm** - Fast, disk-efficient package manager

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm 8+

### Installation

# Clone the repository
git clone https://github.com/bel7aG/comy.ai.git

# Navigate to the project
cd boilerplate

# Install dependencies
pnpm install













# Start the development server
pnpm dev
Open http://localhost:3000 to view the app.
